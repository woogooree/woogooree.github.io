## SELECT

- SELECT 절 / WHERE 절 / GROUP BY 절 / ORDER BY 절로 나눌 수 있다.
- SELECT 다음에는 속성 + FROM 

```
SELECT 속성명 FROM 테이블명
// 속성명 다음에 as 별칭명과 그룹함수(COUNT(속성), SUM(속성) 등) 사용가능
WHERE 조건문
GROUP BY 속성명 HAVING 조건문
ORDER BY 속성 ASC(오름차순) / DESC(내림차순)
```
